Algorithm Explanation

Reed-Muller Codes and RPA Decoding

The RPA (Recursive Projection Aggregation) algorithm decodes Reed-Muller codes through these steps:

  1. Projection: Project received vector \(y\) onto cosets of \(B_i=\{0, z_i\}\) for each non-zero vector \(z_i\)
  2. Recursive Decode: Decode each projection using a simpler decoder (FHT for base case)
  3. Aggregation: Combine decoded results using majority vote to estimate the codeword
  4. Iteration: Repeat until convergence or max iterations reached

The visualization shows a single iteration of this process for \(RM(3,2)\) code.
